# Shishu Sneh – Smart Child Healthcare Monitoring App 👶📱

Shishu Sneh is a smart Android healthcare monitoring application developed using Kotlin and Firebase. The application helps parents digitally manage child healthcare information including vaccination reminders, growth tracking, feeding schedules, milestone monitoring, and emergency healthcare support.

The project aims to improve digital healthcare accessibility for children and assist parents in maintaining healthcare records efficiently through a secure and user-friendly mobile platform.

---

# 📌 Problem Statement

Many parents face difficulties in tracking their child’s healthcare activities manually. Important records such as vaccination schedules, nutrition tracking, growth monitoring, feeding routines, and healthcare history are often unorganized and difficult to manage.

Shishu Sneh solves this problem by providing a centralized digital healthcare monitoring system using modern Android technologies and Firebase cloud integration.
