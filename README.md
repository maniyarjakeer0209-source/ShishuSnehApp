# Shishu Sneh – Smart Child Healthcare Monitoring App 👶📱

Shishu Sneh is an Android application developed using Kotlin and Firebase to help parents monitor and manage their child’s healthcare information digitally.

## 🚀 Features

- 🔐 Firebase Authentication Login
- 👶 Add Child Details
- 🏥 Child Health Monitoring
- 💉 Vaccine Reminder Section
- 📈 Growth Tracking
- 🍼 Feeding Tracker
- ⭐ Milestone Tracking
- 📋 Profile Management
- ☰ Navigation Drawer
- 🌙 Dark Mode Settings
- 🔔 Notification & Reminder Module
- ☁️ Firebase Firestore Integration

## 🛠️ Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Android Studio
